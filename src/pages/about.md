# Hi, I'm apacheli.

I'm a full-stack web developer and graphic designer from Milwaukee. I build open-source software with Bun and
TypeScript.

Check out [my blog](/blog) to see what I've been up to.

## How It Started

Choices for Discord bots were pretty limited back in 2015. There was this bot called _AutoModerator_ which you could
invite via direct messaging. It did what it needed to do, but we needed something custom as our squad grew.

That's when a friend introduced me to programming. We were copying and pasting code from the internet without really
understanding what it did. To be fair, we were still middle schoolers at the time.

## Hobbies

I like to produce music, play video games, and occasionally dabble in digital art.
